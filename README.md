# Store-Sales-Time-Series-Forecasting---ML-Regression-Project

This project focuses on predicting retail store sales using machine learning and time series forecasting techniques.

---

## 📌 Project Overview
The goal of this project is to analyze historical sales data and build a model that can predict future sales.  
It helps businesses make better decisions for inventory, promotions, and sales planning.

---

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit

---

## 🚀 Features
- Data preprocessing and analysis
- Machine learning model for sales prediction
- Interactive Streamlit web application
- Dashboard with basic metrics and charts
- User input-based prediction system

---

## 📊 Project Structure
- `Regression_Time_Series.ipynb` → Model building & analysis  
- `app.py` → Streamlit web application  
- Dataset files → Training & testing data
